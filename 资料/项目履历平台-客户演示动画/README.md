# 项目履历平台客户演示动画

## 文件

- `index.html`：客户演示动画页面。
- `DESIGN.md`：动画视觉风格说明。
- `renders/项目履历平台-客户演示动画.mp4`：已导出的标准版演示视频。
- `renders/项目履历平台-客户演示动画-draft.mp4`：草稿版演示视频。

## 打开方式

直接用浏览器打开：

`E:\HTML\AI赋能参赛\资料\项目履历平台-客户演示动画\index.html`

## 播放控制

- 空格：暂停 / 继续
- R：重新播放

## 演示内容

动画按 8 个段落讲解平台：

1. 平台定位：让项目过程可追溯、可交接、可复盘。
2. 业务痛点：需求散落、发包难追、交接靠口头。
3. 首页总览：项目数量、人员、需求、发包与风险展板。
4. 项目履历：按业务发生时间记录项目过程。
5. 需求与发包关联：发包能反查解决了哪些需求。
6. 资料与环境：Git、文档、中间件配置沉淀。
7. 人员交接：个人项目资料和交接状态。
8. 价值收束：把经验从人脑沉淀到系统。

## 注意

页面通过 CDN 加载 GSAP 动画库。客户演示机器需要能访问外网；如果要完全离线演示，后续可以把 GSAP 文件下载到本地并改为本地引用。

已在本机补充 portable FFmpeg 到 `E:\HTML\AI赋能参赛\tools\ffmpeg\`，仅用于本项目导出视频，不修改系统环境变量。
